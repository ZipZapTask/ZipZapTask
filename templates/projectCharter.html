Objectives

  • • Create a responsive and intuitive to-do list application
  • • Implement core task management features
  • • Ensure high performance and reliability
  • • Deliver a polished user interface

Communication Standards

The main form of communication will be through Microsoft teams, email will be used for private communication or if you need to contact an individual.

Key Features

  • • Task creation, editing, and deletion
  • • Priority levels (High, Medium, Low)
  • • Due date assignment and tracking
  • • Task categorization system
  • • Push notifications and reminders
  • • Responsive and intuitive user interface

Technical Specifications

  • • Backend: Python with Flask framework
  • • Database: SQLite for data persistence
  • • Frontend: HTML, CSS, JavaScript
  • • Version Control: MS Teams History/Git

Main Project Success Criteria

  • • All key features implemented and functioning correctly
  • • Application performs without significant bugs
  • • User interface is intuitive and responsive

Approach

  • • Develop a detailed work breakdown structure, scope statement, and Gantt chart within one month to outline the tasks required for project completion.
  • • Hold weekly progress review meetings with the core project team.
  • • Conduct thorough software testing based on approved test plans.
